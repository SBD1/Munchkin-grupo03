## 🧝 Como jogar?

O projeto conta com um makefile. Para rodar o projeto, o pacote [make](https://www.wikiwand.com/en/Make_(software)) deve estar instalado na máquina.

'''sh
docker network create munchkin
'''

'''sh
make up
'''

'''sh
make migrate
'''
